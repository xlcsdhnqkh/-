# Transformer IMDB Sentiment Analysis

## 项目简介

本项目实现了基于 Transformer 的 IMDB 情感分类任务，
包含手写 Self-Attention 与预训练模型推理两部分。

## 环境说明

Python 3.8+
PyTorch
Transformers
